# AI 读书视频自动化生产 Agent — 设计文档

> 版本: v1.0 | 日期: 2026-06-01 | 状态: 设计完成

## 1. 项目目标

构建一个稳定的自动化 Agent，每天定时生产 AI 读书类视频，模仿 B站 up 主"七页读书吧"（65.5 万粉丝，537 个视频）的内容风格。V1 目标是快速跑通整条链路，稳定产出可用视频。

## 2. 视频类型

| 类型 | 时长 | 频率 | 内容 |
|------|------|------|------|
| 金句短视频 | 30-60s | 每天 1 支 | 书摘金句 + 一句话书评 |
| 睡前阅读长视频 | 10-25min | 每周 1 支 | 全书精读/漫谈/沉浸式朗读 |
| 作文素材短视频 | 30-70s | 每周 2-3 支 | 写作技巧 + 例句 |

每天生成 3 个候选视频，质量审核后选最优 1 个发布。

## 3. 架构总览

三层架构：调度层 → 编排层 → 执行层

```
调度层: cron / launchd 定时触发
编排层: orchestrator.py 总控管线
执行层: crawler/ content/ media/ video/ review/
```

## 4. 模型选型

| 用途 | 模型 | 选用理由 |
|------|------|---------|
| 文案润色 + 质量评估 | **kimi-k2.6** | 中文文学创作口碑最佳，文笔优美，情感渲染强，适合书评润色和金句提炼 |
| AI 生图 | **qwen-image-2.0-pro** | 文字渲染自然，组图风格一致性好，指令遵循强，适合封面+书名叠加 |
| TTS 语音合成 | **qwen3.5-omni-plus** | 多模态模型，语音合成自然，支持音色调节 |
| 视觉理解（审核用） | **qwen3.5-omni-plus** | 图片/视频内容理解，用于图文匹配度审核 |

## 5. 模块设计

### 5.1 爬虫模块 (src/crawler/)

**设计原则：** 增量积累 + 反爬策略 + 结构化存储

- **数据源：** 豆瓣读书 + 微信读书
- **爬取节奏：**
  - 每日：爬取 5-10 本热门/新书的基本信息
  - 每周：深度爬取 2-3 本书的高赞书评
  - 增量：只爬新数据，已爬过的跳过
- **存储：** SQLite 本地数据库
  - `books` 表：id, title, author, isbn, cover_url, douban_id, category, ...
  - `reviews` 表：id, book_id, source, rating, content, highlights, crawled_at, ...
  - `crawl_log` 表：id, source, task_type, crawled_at, status, ...
- **反爬措施：** User-Agent 轮换、请求间隔 3-5s 随机延迟、失败重试 3 次指数退避

### 5.2 多模态工具模块 (src/media/)

统一封装 API 调用，屏蔽底层细节，供管线各步骤调用。

- **client.py** — `QwenClient` 统一客户端，封装 API key/base_url、自动重试、超时处理
- **image.py** — `text_to_image(prompt, style, size)` / `batch_generate(prompts[])` → 返回本地路径
- **tts.py** — `text_to_speech(text, voice, speed)` → 返回音频文件路径，支持长文本分段合成+拼接
- **vision.py** — `analyze_image(image)` / `compare_text_image(text, image)` → 用于审核

### 5.3 内容生成模块 (src/content/)

**选书策略 (selector.py)：**
- 从本地 SQLite 中按策略选书
- 支持：随机多样化 / 品类轮换 / 金句密度优先
- **避免 30 天内重复选用同一本书**
- 候选池：每次选 3 本，各自生成视频后择优

**脚本润色 (polisher.py)：**
- 输入：爬取的书评原文 + 金句
- 调用 kimi-k2.6 润色
- 三种输出模板：

```
短视频模板：
  片头句: "[每天一个顶级文笔] 今天分享的是《{书名}》"
  主体句: "{一句金句/书评}"
  片尾句: 无

长视频模板：
  片头句: "睡前听完一本书，今天读的是《{书名}》"
  主体: 按书籍类型选择结构
    · 小说 → 情感沉浸式（3-5 个情感段落）
    · 干货 → 精读式（章节提炼 + 金句）
    · 社科 → 主题漫谈式（3-5 个核心主题）
  片尾句: "{一句收尾感悟}"

作文素材模板：
  片头句: "{如何在作文里写XX}"
  主体: 3-4 个写作角度 + 例句
  片尾句: "{一句总结}"
```

### 5.4 视频合成模块 (src/video/)

**合成流程：**

1. **片头生成 (intro.py)**
   - 固定背景图 + 模板文字叠加
   - 书名封面快闪：3-5 张其他封面 (0.3s/张) → 停在本期封面
   - 时长：视频总时长的 5%-8%

2. **主体合成 (composer.py)**
   - 输入：图片列表 + 音频文件 + 脚本时间轴
   - 图片轮播：每张 4-6 秒，带缩放/平移效果 (Ken Burns)
   - **字幕：中英双语**，SRT 格式，底部居中，逐句显示
   - **画面内容：书籍封面或 AI 生成插画，叠加书名 + 作者（格式：xx/著）**
   - 背景音乐：轻音乐低音量垫底（可选）

3. **合成引擎：FFmpeg**
   - 图片序列 → 视频流 + 音频流 → 最终 mp4
   - 分辨率：1080p (1920x1080)

**图片-脚本对应关系：**
脚本由 LLM 分段，每段配一张生图 prompt：
> "根据这段文字生成一张配图：{段落文本}，风格：{风格描述}"

**预估资源：**
- 短视频 (40s)：8-10 张图 + 1 段音频 → ~2 分钟合成
- 长视频 (15min)：60-100 张图 + 1 段音频 → ~5 分钟合成

### 5.5 质量审核模块 (src/review/)

**审核流程：** 每次生成 3 个候选视频 → 自动评分 → 选最高分。最高分 < 阈值则全部丢弃，下一轮重试。

**评分维度：**

| 维度 | 权重 | 评估方式 |
|------|------|---------|
| 文案质量 | 30% | kimi-k2.6 评分 |
| 语音自然度 | 20% | qwen3.5-omni-plus 评估 |
| 图文匹配度 | 25% | qwen3.5-omni-plus 对比 |
| 结构完整性 | 15% | 规则检查（时长/分段/格式） |
| 情感吸引力 | 10% | kimi-k2.6 多维度打分 |

**阈值策略：**
- ≥ 80 分：直接保留，可发布
- 60-79 分：标记"需人工审核"
- < 60 分：丢弃

### 5.6 调度模块 (src/scheduler/)

**每日流程（cron 触发）：**

| 时间 | 任务 |
|------|------|
| 08:00 | 爬虫增量任务（爬取新书信息） |
| 10:00 | 选书 → 生成 3 个短视频候选 |
| 12:00 | 质量审核 → 选最优短视频 |
| 14:00 | 每周四：长视频生成（耗时长） |
| 18:00 | 每周四：长视频审核 |
| 20:00 | 输出当日最终视频到 data/output/ |

**实现：** macOS launchd plist 或 crontab，每个步骤失败自动重试 3 次后告警。

**告警方式：** 发送邮件到 `luilsn0501@gmail.com`。

## 6. 语音旁白设计

基于对"七页读书吧"实际视频的 qwen3.5-omni-plus 分析：

| 参数 | 短视频 | 长视频 |
|------|--------|--------|
| 音色 | 温暖沉稳男声 | 同，更低沉 |
| 语速 | 0.8-0.9x | 0.7-0.8x |
| 音高 | -1 到 -2 | -1 到 -2 |
| 情感 | 娓娓道来，克制但有感染力 | 低声诉说，深沉内敛 |
| 停顿 | 句尾延长 30%-50% | 句尾延长 50%+，段落间更长 |
| 后处理 | 轻微混响 | 轻微混响 |

## 9. 非功能需求

- **可靠性：** 每个步骤失败自动重试 3 次，失败后邮件告警
- **可扩展性：** 新增视频类型只需添加模板配置，不改管线
- **可观测性：** 每步记录日志，爬取记录写入 crawl_log 表
- **本地运行：** 所有依赖在本地，无需 Docker

## 10. V1 范围

**包含：**
- 爬虫：豆瓣 + 微信读书增量爬取，SQLite 存储
- 内容：选书 + LLM 润色 + 三种视频模板
- 素材：AI 生图 + TTS 语音 + 封面爬取
- 合成：FFmpeg 视频合成 + 片头快闪 + 中英双语字幕
- 审核：5 维度自动评分 + 候选筛选
- 调度：cron 定时任务 + 邮件告警

**不包含（V2+）：**
- 自动发布到 B站
- AgentScope 框架集成
- Web UI 管理界面
- 多平台分发
- 视频 A/B 测试
